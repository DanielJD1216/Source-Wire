These fixtures are synthetic only.

They model MCP adapter declarations, input validation, and MCP-to-API policy envelopes without adding an MCP server runtime, API server, route handlers, database, deployment, connector, private data, client data, private implementation code, or AGPLv3 code.

Use Node.js 22 with npm from the repository root. For the complete local setup path, read the [Quickstart](../../../docs/quickstart.md).
