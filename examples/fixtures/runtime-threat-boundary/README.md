These fixtures are synthetic only.

They model threat-boundary cases for the public Source-Wire owner-hosted runtime path without adding a real API server, MCP server runtime, database, deployment, connector, private data, client data, private implementation code, or AGPLv3 code.

Use Node.js 22 with npm from the repository root. For the complete local setup path, read the [Quickstart](../../../docs/getting-started/quickstart.md).

Run:
