# Internal Project Archive

This directory preserves Source-Wire planning records, approval packets, implementation proofs, release rehearsals, issue drafts, decision prototypes, and other project-history material.

These files remain public for provenance and review, but they are not the first-run documentation, public API reference, or proof that a hosted production runtime exists.

## Use This Archive When

- tracing why a boundary or release decision was made,
- reviewing the evidence behind a synthetic proof,
- auditing historical approval and stop conditions,
- continuing an explicitly approved project workflow.

## Do Not Use This Archive As

- the onboarding path for a new adopter,
- the current package API reference,
- evidence that a synthetic runtime is a production service,
- permission to deploy, publish, migrate data, or accept contributions.

For current public documentation, start with the [documentation home](../README.md). For the product overview, start with the [project README](../../README.md).
