# API Contract Fixtures

These fixtures are synthetic only.

They model the Source-Wire API policy contract without adding an API server, route handlers, MCP server runtime, database, deployment, connector, private data, client data, private implementation code, or AGPLv3 code.

Use Node.js 22 with npm from the repository root. For the complete local setup path, read the [Quickstart](../../../docs/getting-started/quickstart.md).

Run:

```bash
npm run runtime:api-policy-smoke
```
