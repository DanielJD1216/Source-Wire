# Owner-Hosted Runtime Fixtures

`owner-hosted-runtime-fixture-matrix.json` is synthetic-only proof data for the owner-hosted API and MCP runtime skeleton.

The fixture covers:

- authorized trusted-memory read,
- source evidence search,
- missing capability denial,
- wrong namespace denial,
- MCP policy bypass denial,
- source maintenance without automatic trusted-memory promotion,
- candidate review without promotion,
- owner or application controlled trusted-memory approval,
- cited stale evidence gaps,
- audit metadata preservation.

The fixture must not contain real local paths, emails, domains, tokens, account IDs, source exports, client data, private implementation code, AGPLv3 code, or private proof content.
